Large Language Models Workshop

University of Strasbourg - Spring 2025

by Roman Jurowetzki

Workshop Overview

This workshop explores advanced applications of Large Language Models (LLMs) in analytical contexts. Through hands-on sessions and practical examples, participants will learn to leverage LLMs for various analytical tasks and applications.

Key Topics

  • LLM Batch Processing and Analysis
  • Topic Modeling with BERTopic
  • Structured Output Generation
  • RAG (Retrieval-Augmented Generation) Applications
  • Analytical Use Cases of LLMs

Technical Requirements

Some notes on technical requirements:

  1. We'll use Google Colab for computation, which provides free GPU access for certain tutorials. You'll need a Google account to use this.
  2. Alternatively, you can run everything locally if you prefer and are comfortable with local installations.
  3. Together.ai will be used along with their API for accessing hosted LLMs. By creating an account on their platform, you'll receive $5 in free credits.
  4. Kluster.ai platform is required for batch inference tasks. New accounts receive $5 in free credits with account creation.
  5. A Huggingface account is required for two main reasons: a) to run Argilla's user interface and b) to access models and host your data and models if needed.
