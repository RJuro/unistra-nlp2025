Workshop Materials

This workshop consists of several hands-on sessions where participants will learn to leverage Large Language Models for various analytical tasks. All materials are available as Jupyter notebooks that can be accessed directly or opened in Google Colab.

Day 1 - March 4, 2025

Session 1: Introduction to LLM Use

Learn the fundamentals of working with Large Language Models, including API setup, prompt engineering, and basic applications.

View Notebook Open in Colab

Session 2: Batch Processing with Kluster

Learn how to scale up text extraction by processing large batches of articles efficiently using the Kluster platform. This tutorial covers setting up batch processing jobs and handling multiple API requests simultaneously.

View Notebook Open in Colab Download Data

Session 3: Results Analysis and Evaluation

Master the techniques for processing and evaluating results from batch processing jobs. Learn to handle JSONL outputs, parse complex responses, and organize data into structured formats for analysis.

View Notebook Open in Colab Download Data

Day 2 - March 5, 2025

Session 4: Topic Modeling with BERTopic and Generative AI

Explore advanced topic modeling using BERTopic, combining transformer models with Generative AI to extract meaningful topics from podcast transcripts.

View Notebook Open in Colab

Session 5: Comparing LLMs for Climate Claim Classification

Compare different LLM models on climate change claim classification, using both local (Ollama) and cloud-based models. Learn about inter-rater reliability metrics and randomness testing.

View Notebook Open in Colab View Slides

Day 3 - March 6, 2025

Session 6: RAG Applications with LlamaIndex and Gemini

Learn to implement Retrieval-Augmented Generation (RAG) systems using LlamaIndex and Google's Gemini model to combine LLMs with external knowledge sources.

Download Notebook Open in Colab

Session 7: Text Classification with Argilla and SetFit

Explore advanced text classification techniques using Argilla for data annotation and SetFit for few-shot learning applications.

Download Notebook Open in Colab
